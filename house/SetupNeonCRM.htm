Instructions for using the Guest Transfer facility:

1.      Browse to the HHK Guest Tracking site and log in.

2.      Under the “House” menu item along the top of the page, select “Guest Transfer”.

Once on the Guest Transfer page, you can search for names on either system:

·        NeonCRM Last Name Search:  Enter at least 3 letters of the last name. 

·        Local (HHK) Name search:  Enter at least 3 letters of the first or last name.  When the name comes up, one of two buttons will appear depending upon if the NeonCRM Account Id is set locally.

o   “Update Remote” – click to update Neon.

o   “Transfer to Remote” – click to send this new account to Neon.

Also, you can do bulk transfers of filtered HHK accounts to NeonCRM.

·        Set the Time Period to filter the HHK accounts.

·        Click “Get Records”

·        Click on the “Transfer” button when it appears

 

After clicking on “Transfer”, HHK will automatically transfer any new contacts to NeonCRM.  The Result column indicates what happened:

·        New Contact – record successfully transferred to NeonCRM

·        Previously Transferred – NeonCRM already has this contact.

·        Multiple Remote Matches – There are more than one match for the local contact.

·        Error message – See section on Errors below.

 

Errors

Several errors can occur during the transfer process.  These errors are reported by the NeonCRM API.  Please refer to the NeonCRM API documentation for a description of the error
